## Версионирование

Мы используем [Semantic Versioning](https://semver.org/):
- **MAJOR** - несовместимые изменения API
- **MINOR** - новые функции (совместимо с предыдущей версией)
- **PATCH** - исправления ошибок

## Миграция между версиями

### Обновление с v0.x на v1.0.0
1. Очистите кэш: `localStorage.clear()`
2. Удалите старые файлы плагина
3. Установите новую версию
4. Переконфигурируйте API ключи в новом config.json

## Политика поддержки

- **Текущая версия**: полная поддержка
- **Предыдущая версия**: баг-фиксы только
- **Старые версии**: нет поддержки
